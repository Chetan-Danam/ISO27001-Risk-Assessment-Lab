# ISO 27001 Risk Assessment & Risk Register Lab

This project demonstrates a practical implementation of an ISO/IEC 27001:2022 risk assessment process.

## Objective
To identify organizational assets, analyze security threats and vulnerabilities, calculate risk scores, and map mitigation controls to ISO 27001 Annex A controls.

## Project Components

Asset Register  
List of organizational assets including systems, data, and infrastructure.

Risk Assessment  
Identification of threats, vulnerabilities, likelihood, and impact.

Risk Register  
Documented risks with severity levels and treatment plans.

Control Mapping  
Security controls mapped to ISO/IEC 27001 Annex A.
